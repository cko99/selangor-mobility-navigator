Help & Support

About this project

Selangor Mobility Navigator is an independent portfolio Web GIS prototype for exploring public-transport geography across Selangor and the Klang Valley.

Using the map

Use the map filters to show or hide transport modes. Open Nearest Station and explicitly allow location access to calculate approximate straight-line distances.

Data limitations

Static schedules are not realtime arrivals. Trip results are not official journey recommendations. Fare, service-alert, parking-occupancy and account integrations are not active.
