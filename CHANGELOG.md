# Changelog

## [0.2.1] - 2026-07-21

### Changed

- Improved prototype data disclosure.
- Removed unsupported dummy account and payment information.
- Renamed misleading live-data labels.
- Added professional Coming Soon states.
- Added project and data-status disclosure.
- Improved navigation stability.
- Improved responsive and public-release polish.

### Fixed

- Replaced the misleading Live Map and verified-live presentation with a static Transit Map disclosure.
- Removed dashboard metrics that had no supported values.
- Made trip-planner success and empty states match the actual visible result.
- Removed inert trip, schedule, notification, saved-location, payment and feedback controls.
- Added explicit geolocation guidance, bounded radius handling and straight-line distance wording.
- Added accessible labels, focus treatment, reduced-motion support, a favicon and a web manifest.
- Preserved visible OpenStreetMap attribution and removed unconfigured Satellite and Hybrid controls.

### Removed

- Dummy masked payment card and payment-method action.
- Fake account avatar, travel statistics and history containers.
- Unsupported on-time, average-wait and network-load cards.
- Fake subscription, upload, feedback-success and settings interactions.
- Unsupported global alert badge and red notification indicator.
- Unconfigured Satellite and Hybrid basemap choices.
