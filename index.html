The Context

Schematic vs. Geographic Reality

Schematic Transit Diagram

  • Useful for network relationships
  • Simplified geography
  • Harder to understand real station surroundings
  • Does not directly show the user’s current location

Geographic Web GIS

  • Real-world location context
  • Station discovery
  • Nearby-transit analysis
  • Route and station information
  • External navigation options

Workflow

How It Works

Open the application

Launch the Web GIS interface in your browser.

Allow location access

Allow location access to find nearby stations.

Select a station

Select a station or nearby transport point on the map.

Open navigation

Open navigation or explore route information.

Privacy Note: Location is requested only when the user chooses to use location-based features. Location data is not stored.

Prototype Coverage
